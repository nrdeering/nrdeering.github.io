Mission to Mars

Web-scraping application that collects
data from NASA and other web sources
containing information and updates
on the exploration of Mars.

Get Started

Objective

Build a web application that scrapes various websites for data related to the Mission to Mars and exports the data to a Mongo database. Complete initial scraping using Jupyter Notebook, Python, BeautifulSoup, Pandas, and Requests/Splinter. Sources of scraping include Nasa's Mars Exploration Program, Nasa's Jet Propulsion Laboratory, Mars Weather's Twitter page and the Space Facts website. Once Scraped, use MongoDB with Flask templating to create a new HTML page that displays all of the information that was scraped from the URLs above (This step is not yet complete and will be coming soon).

Code

First, import the dependencies and make a connection to a Mongo database. Next, grab the URL of the page that is going to be scraped and make a connection. You should see 'Response [200]' in the output if the connection is succesful.

After making a connection to the page, use BeautifulSoup to get the web page HTML. Next, use soup.prettify() to view how the different tags are nested, making it easier to find the elements for scraping.

Next, find all div elements assigned the class 'slide'. Iterate through the 'results' and find the titles and descriptions of the articles on the page.

The first URL is scraped and added to the Mongo database! Next, make a connection with the next web page and print out the HTML of the page.

After making a connection to the URL and getting the HTML code, find the elements linked to the feature image, output it's URL and add it to the MongoDB.

The second URL is scraped and added to the Mongo database! Next, make a connection with the next web page and print out the HTML of the page.

After making a connection to the URL and getting the HTML code, find the elements that are linked to the Tweets that specifically detail the weather on Mars. Find the most recent Tweet that gives an update of the weather on Mars(Hint: all Tweets that give updates of the weather on Mars start with 'Insight Sol'). Any Tweet that did not start with the string 'Insight Sol' was skipped. Once the Mars weather Tweets are scraped, print out the first Tweet in the list.

The Tweet was found an added to the MongoDB! Finally, make a connection to the final URL and get the HTML code.

Once a connection is made, find the table that holds the attributes of Mars. Output the table to a dataframe and add to the Mongo database.

Convert the dataframe above to HTML.

Output all items scraped.

Conclusion

BeautfulSoup is a powerful tool to find information and data within HTML code that would take many hours to find. Finding the correct elements required quite a bit of problem solving and troubleshooting to make sure the functions within the program were scraping and collecting the correct pieces of data. The MongoDB that was created will eventually be implemented into a Flask app that loads the database into an HTML page, so stay tuned!
